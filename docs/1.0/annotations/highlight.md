---
title: Highlight
description: The highlight annotation will apply a background color to the line(s) it is applied to.
breadcrumbs: [Documentation, Annotations, Highlight]
---

```php
// lighty {"skipLineParsing": true}
return [
    'extensions' => [
        // Add attributes straight from markdown.
        AttributesExtension::class,
        // Add Lighty syntax highlighting.[lighty highlight]
        LightyExtension::class,[lighty highlight]
    ]
]
```
